# Los Tules Website Editing Guide

This steering file provides context for editing the Los Tules Mexican Restaurant website.

## Project Structure

```
site/
├── app/
│   ├── page.tsx          # Main website content (ALL sections)
│   ├── layout.tsx        # Page metadata and HTML wrapper
│   └── globals.css       # Global styles and Tailwind directives
├── components/ui/        # Reusable UI components (Button, Card)
└── out/                  # Build output (generated by npm run build)

infra/
├── app.py                # CDK app entry point (uses InfraStack)
├── infra_stack.py        # CDK infrastructure definition (ACTIVE - both buckets)
├── stacks/
│   └── static_site_stack.py  # Old stack (NOT USED - single bucket only)
├── los-tules-menu2026.pdf # Menu PDF file
└── requirements.txt      # Python dependencies
```

## Important: Stack Configuration

The project uses `InfraStack` from `infra_stack.py` which deploys:
1. Website S3 bucket (private, via CloudFront)
2. Menu PDF S3 bucket (public, named `los-tules-menu-files`)

The old `StaticSiteStack` in `stacks/static_site_stack.py` is no longer used.

## Common Editing Tasks

### Update Restaurant Hours
Edit the HOURS constant (lines 29-37):
```javascript
const HOURS = [
  { day: "Tuesday", time: "11:00 AM – 10:00 PM" },
  // ... update times here
];
```

### Change Phone Number
Update in two places:
- Line 416: Contact section
- Line 479: Footer

### Update About Text
Edit paragraphs in lines 285-295

### Change Gallery Images
Edit GALLERY_ITEMS array (lines 14-27):
```javascript
{ label: "New Image", image: "https://images.unsplash.com/..." }
```

### Update Menu PDF
1. Upload new PDF to S3 bucket
2. Update URL in lines 260 and 341

## Build and Deploy

### Local Testing
```bash

```

### Build for Production
```bash

# Creates site/out/ directory
```


## Styling

- **Global styles**: `site/app/globals.css`
- **Component styles**: Inline Tailwind classes in JSX
- **Colors**: 
  - Primary: Teal (teal-600, teal-700)
  - Accent: Amber (amber-400, amber-500)

## Important Notes

- This is a Next.js static site (output: "export")
- All images are from Unsplash (external URLs)
- No server-side rendering or API routes
- Menu PDF is hosted separately in S3
- CloudFront serves the static files

## When Helping Users

1. **For content changes**: Direct them to specific line numbers in `page.tsx`
2. **For styling changes**: Modify Tailwind classes or `globals.css`
3. **For structure changes**: Explain JSX component structure
4. **For deployment**: Use `./deploy.sh` or CI/CD pipeline
5. **For testing**: Always recommend `npm run dev` first

## User Skill Level

The user is not experienced with React or web development. When helping:
- Provide exact line numbers
- Show before/after code examples
- Explain what each change does
- Avoid complex React concepts
- Focus on practical, copy-paste solutions
